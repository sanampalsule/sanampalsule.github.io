JOB SEEKER and EMPLOYER PORTAL

This project aims to develop a comprehensive platform for job seekers and employers, focusing primarily on the backend implementation using Spring Boot and MongoDB. By leveraging Spring Boot for backend development and MongoDB for data storage, the project endeavors to provide a seamless experience for users in searching for job profiles, posting job requirements, and connecting job seekers with employers.

Technology Stack Used:

  • Spring Boot
  • MongoDB
  • React
  • Implement features such as job searching, posting, and browsing using Spring Boot and MongoDB backend, ensuring efficient data management and user interaction.
  • Utilize React frontend to seamlessly integrate with the backend, providing a scalable and user-friendly job portal experience.
